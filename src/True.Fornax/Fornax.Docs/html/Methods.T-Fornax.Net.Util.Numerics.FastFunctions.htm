FastFunctions Methods

The FastFunctions type exposes the following members.

Methods

  Name Description
Asin
Returns the arc sine of a value.

The returned angle is in the range -pi/2 through pi/2. Error is around 1E-7.

Special cases:

  • If the argument is Double or its absolute value is greater than 1, then the result is Double.
Cos
Returns the trigonometric cosine of an angle.

Error is around 1E-15.

Special cases:

  • If the argument is Double or an infinity, then the result is Double.
EarthDiameter
Return an approximate value of the diameter of the earth at the given latitude, in kilometers.
