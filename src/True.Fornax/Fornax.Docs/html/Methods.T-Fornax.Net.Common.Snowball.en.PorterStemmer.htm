Reset
Reset()()()() resets the stemmer so it can stem another word. If you invoke the stemmer by calling Add(Char) and then Stem(String), you must call Reset()()()() before starting another word.
Stem(String)
Stem a word provided as a String. Returns the result as a String.

ToString
