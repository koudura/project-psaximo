WordReader..::..GetWordSet Method (TextReader, Boolean)

Gets the word set, from a Reader TextReader and adds every line as an entry to a HashSet. (ommiting leading and trailing whitespaces).

NOTE: Every Line of the file should contain a word, if isBias is set to false then all then words are preprocessed before entry. Preprocessing includes: lowercasing and removing illegal whitespaces.
