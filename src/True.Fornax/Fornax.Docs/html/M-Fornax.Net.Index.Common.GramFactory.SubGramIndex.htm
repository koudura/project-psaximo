GramFactory..::..SubGramIndex Method

Retrieves a sub index of the given k-gram from a similar Gram-Index. The "similar" gram-index is determined by the size of the grams in the index. e.g In other to avoid retrieval mismatch:
k-gram of size (2) should be the input into a gram-index of gram-size (2).
