Members

Member nameDescription
Minimal The Minimal approach rules :

No page retrieval scoring and ranking.Only gets a [n] number of pages (where [n] is a specific threshold value that indicates the upper bound of the frontier. [Default = 10]).

Only gets pages relative to the absolute link/page as inputed by user.this option doesn't initiate branching from the hostname.

Normal The Normal retrieval approach rules :

Page retrieval scoring and ranking.Only gets a [n] number of pages (where [n] is a specific threshold value that indicates the upper bound of the frontier. [Default = 15]).

Only gets pages relative to the absolute link/page as inputed by user.this option doesn't initiate branching from the hostname.

Detailed The Detailed retrieval approach rules :

Page retrieval scoring and ranking.Only Traverses the web graph and retrieves the top ranked documents relative to a link. (Where the top-ranked is determined by fornax.net, NOTE: the resulting pages may span a huge result.) Traversion of pages also include Host-name as root. i.e the hostname serves as root for this traversion. NOTE: This occurs on a separate thread.

Absolute The Absolute retrieval approach rules : page retrieval scoring and ranking.Traverses the web graph of the given link to the lowest depth.
