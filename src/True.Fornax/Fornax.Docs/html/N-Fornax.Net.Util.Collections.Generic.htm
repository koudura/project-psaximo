LurchTable<(Of <(<'TKey, TValue>)>)>
LurchTable stands for "Least Used Recently Concurrent Hash Table" and has definite similarities to both the as well as Java's LinkedHashMap. This gives you a thread-safe dictionary/hashtable that stores element ordering by insertion, updates, or access. In addition it can be configured to use a 'hard-limit' count of items that will automatically 'pop' the oldest item in the collection.
