Fornax.Net.Analysis.Tools Namespace

The Tools namespace.

Classes

  Class Description
BufferTrie
A trie data structure holding a bufffer of characters. i.e strings are inserted by character precedence per text-input. Inputs include [character] array and text as [string].
ClusterTrie
A trie data structure holding a cluster of characters. i.e strings are inserted as phrases. Input includes phrase and sentence strings.
FornaxLevenshteinEdit
The Fornax.Net specific levenshtein edit distance class.
FornaxStemmer
The Stemmer class transforms a word into its root form. Implementing the a modified porter Stemming Algorithm
Ngram
N-Gram representation of string text.
Soundex
Soundex Code object. Used for phonetic query matching and mapping.
Synset
Represents a Synonym set for a given word retrieved from a synonym index. NOTE: If Synonym Index isnt provided and there be no existing index, the defaut @Default would be used.
