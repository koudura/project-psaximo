Free The Free Text or multi-term Query.
e.g "Fornax is awesome";
KeyWord The keyword or single-term query e.g "Awesome"

NOTE: should not be mistaken for Phrase query.

Boolean The boolean query type. e.g "mit AND apache".
Phrase The phrase query. e.g "\"America\""
Zone The zone query type. e.g "title:query".
Frequency The frequency query type. e.g "america >30"
Proximity The proximity query type. e.g "check $1 mate"[adjacency] , "check %2 mate"[fuzzy]
Regex The regex query type. e.g "%d+[querystring]*"
WildCard The wild card query. e.g "mo*e*" => {money,morsel,...}
Truncated The truncated query type. i.e "term?" where '?' is the truncation symbol.
