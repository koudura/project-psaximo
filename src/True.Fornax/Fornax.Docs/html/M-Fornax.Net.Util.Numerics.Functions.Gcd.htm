Remarks

NOTE: A greatest common divisor must be positive, but 2^64 cannot be expressed as a Int64 although it is the GCD of Int64 and 0 and the GCD of Int64 and Int64. So in these 2 cases, and only them, this method will return Int64. see http://en.wikipedia.org/wiki/Binary_GCD_algorithm#Iterative_version_in_C.2B.2B_using_ctz_.28count_trailing_zeros.29
