EquatableList<(Of <(<'T>)>)>..::..Equals Method

Overload List

  Name Description
Equals(IList<(Of <<'(T>)>>))
Compares this sequence to another IList implementation, returning true if they are equal, false otherwise.

The comparison takes into consideration any values in this collection and values of any nested collections, but does not take into consideration the data type. Therefore, EquatableList<(Of <(<'T>)>)> can equal any IList with the exact same values in the same order.

Equals(Object)
If the object passed implements IList, compares this sequence to obj, returning true if they are equal, false otherwise.

The comparison takes into consideration any values in this collection and values of any nested collections, but does not take into consideration the data type. Therefore, EquatableList<(Of <(<'T>)>)> can equal any IList with the exact same values in the same order.
