Remarks

The equality checking of collections will recursively compare the values of all elements and any nested collections. The same goes for using Object - the string is based on the values in the collection and any nested collections.

Do note this has a side-effect that any custom ObjectObject, and Object implementations for types that implement IList (including arrays), ISet, or IDictionary will be ignored.
